## Beat Script

When proposing a scene, lay it out:

```
## [Scene title — one evocative phrase]

**Enjeu**: [what's at stake — one sentence]
**Bascule**: [what changes by the end — one sentence]

1. [BEAT] — [action]
   → [invisible: want/feel/hide]

2. [BEAT] — [action]
   → [shift/escalation/revelation]

3. [BEAT] — [the turn]
   → [choice/cost/consequence]

**Sortie**: [last image — propels into next scene]
```

3-7 beats. Fewer = moment. More = you're writing it for them.
