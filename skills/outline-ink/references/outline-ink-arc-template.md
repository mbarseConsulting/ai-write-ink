# outline-ink — Arc Template

> Loaded by: outline-ink with `--arc` flag
> Scope: structural spine of a single book — acts, turning points, throughlines
> Prerequisite: read saga outline if it exists before building
> Build section by section. Do not generate all at once.

---

## Agent instruction

Verify consistency with the saga level before building.
The arc must move the series-level throughline (if a saga exists).
Flag any turning point that isn't prepared by what precedes it.
The % positions below are conventions for 3-act commercial fiction — adjust per genre (romance: 4-act, mystery: revelation-driven, literary: non-linear). Label any deviation explicitly.
