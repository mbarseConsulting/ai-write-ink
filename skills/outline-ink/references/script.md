# Scene Script Template

> Used by: outline-ink (`--script`) to produce the scene brief passed to ink
> Ink handles total word count (scene: 800–5000w, chapter: 3000–6000w)
> Beat pacing: `[brief]` = pass through / `[develop]` = give it room / `[anchor]` = center of gravity, expand

---

## [Scene N° — working title]

### Header

- **Location:** [INT/EXT — precise place, not just "a room"]
- **Moment:** [time of day, light, atmosphere — one line]
- **Characters present:**
  - [Name] — wants: [from this scene] / hides: [in this scene] / enters: [physical + emotional state]
  - [Name] — wants: / hides: / enters:

### Function

- **Narrative role:** [what this scene DOES for the story — one sentence]
- **Stakes:** [what can be won/lost, for whom]
- **Change:** [what's different at exit vs entry — for at least one character]

### Execution

- **POV:** [character — mode: `third-limited` / `third-omniscient` / `first-person` / `free-indirect`]
- **Register:** [combine as needed: `dense` / `sparse` / `dialogue-heavy` / `action-driven` / `interior` / `lyrical`]
- **Tone:** [emotional color — e.g. menacing, tender, ironic, clinical, feverish]
- **Sensory focus:** [what to emphasize: visual / tactile / sound / all]
- **What to show vs tell:** [explicit call — e.g. "show the fear through body, tell the backstory in one line"]

### Beats

1. `[brief/develop/anchor]` **[BEAT label]** — [action]
   → subtext / what the character hides here
   → anchor line (1 line of dialogue or action that MUST appear)

2. `[brief/develop/anchor]` **[BEAT label]** — [escalation or shift]
   → subtext
   → anchor line

3. `[anchor]` **[BEAT label — the turn]** — [choice / cost / revelation]
   → subtext
   → anchor line 1
   → anchor line 2 *(anchor beats may have 2-3 anchor points)*

*(3–6 beats. Fewer = a moment. More = you're writing it for them.)*

### Exit

- **Last image or last line:** [what closes the scene — concrete, not abstract]
- **Character states at exit:** [physical + emotional — what they carry into the next scene]
- **Bridge:** [one line on what this triggers next]
