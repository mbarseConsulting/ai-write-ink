# outline-ink — Chapter Template

> Loaded by: outline-ink with `--chapter` flag
> Scope: chapter plan — scenes, function, pacing, opening/closing states
> Prerequisite: read arc outline if it exists before building
> Build section by section. Do not generate all at once.

---

## Agent instruction

Verify consistency with the arc level before building.
Every scene in this chapter must serve the chapter's function.
Every chapter must serve a structural function in the arc.
Flag any scene that doesn't earn its place.
