This is a bird's-eye view. Keep it short. No prose paragraphs — tables and bullet points only.
For each work: core conflict + structure skeleton + structural reversal + resonance.
**Depth guard:** if a work's summary exceeds ~10 lines → it needs a dedicated `outline-ink-saga`. This section is a summary, not an outline.
If the author doesn't know yet → "to define" and move on.
Link to detailed saga outlines (`outline-ink-saga`) for depth.
