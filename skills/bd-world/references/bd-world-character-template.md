# bd-world — Character Template

> Loaded by: bd-world with `--character` flag
> Usage: creating a complete character sheet for writing use
> Canon: mark `[CANON]` / `[OC]` if working in an existing universe

---

## Agent instruction

Build the sheet section by section with the author.
Start with identity, then move down toward psychology. Don't skip sections — each layer informs the next.
LANGUAGE section: handle with particular care. This is what differentiates characters on the page.
If the author doesn't know yet → "to define" and continue. Do not invent.

### Quick checklist

- [ ] Facade and core are distinct and not contradictory
- [ ] Wound generates lie, lie generates want, need is distinct from want
- [ ] Language is differentiable from that of other main characters
- [ ] Emotions have concrete physical signals (not "he felt sad")
- [ ] Defense mechanisms match the established wound
- [ ] Writing rules are specific enough to guide an agent
