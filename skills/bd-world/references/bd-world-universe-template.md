# bd-world — Universe Template

> Loaded by: bd-world with `--universe` flag
> Usage: building or extending a universe bible, generic for any fictional universe
> Canon: mark `[CANON]` / `[OC]` for each element if working in an existing universe

---

## Agent instruction

Build section by section with the author. Do not generate all sections at once.
For each section: ask the questions, read the answers, fill in, move to the next.
If the author doesn't know yet → write "to define" and continue.
