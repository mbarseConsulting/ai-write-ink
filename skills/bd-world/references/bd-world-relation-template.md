# bd-world — Relation Template

> Loaded by: bd-world with `--relation` flag
> Usage: creating a relation sheet between two characters for writing use
> Prerequisite: individual character sheets recommended but not required

---

## Agent instruction

Build the sheet section by section with the author.
Work both directions of each section: how A sees B, how B sees A.
Relations are asymmetric by nature — don't look for symmetry.
If the author doesn't know yet → "to define" and continue.
